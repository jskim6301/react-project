모킹(Mocking)이란?
Mock(모의 데이터)을 만들어서 활용하는 방식

통상적으로 data fetch를 해야하는 경우 통신을 통해 응답을 내려주는 서버가 있어야 함

서버가 없는 경우,
api 요청으로 내려올 데이터를 프론트에서 모킹하거나
서버의 역할을 해주는 무언가가 필요

대표 라이브러리 : msw

브라우저의 요청이 서버까지 가지 않고 요청을 인터셉트 하여 msw가 요청을 응답하는 라이브러리

노드환경에서는 브라우저의 서비스워커환경이 없기 때문에 따로
노드에 셋업을 한다 

mocks 폴더에서 
server.js   => 테스트를 위한 폴더 
browser.js  => 브라우저에서 쓸때의 폴더

브라우저에서 Service worker를 활용하는 것이 msw 라이브러리의 특징

Rest API / GraphQL => 모두 모킹가능
